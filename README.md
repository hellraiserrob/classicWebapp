# Webpack seed
A simple development and build seed using webpack

### Features
- webpack
- webpack-dev-server
- eslint
- babel
- pug templates
- mocha / chai test

### Scripts
- npm run dev 
    - starts development server
- npm run build
    - produces a production build
- npm run demo
    - serves up the contents of the build for a demo
- npm run test
    - runs the mocha / chai tests
